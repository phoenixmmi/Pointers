0x00 - 1B

1 Byte  = 8 bit;
1 kByte = 1024 B  = 2^10 B;
1 MByte = 1024 kB = 2^20 B;
1 GByte = 1024 MB = 2^30 B;
1 TByte = 1024 GB = 2^40 B;
1 PByte = 1024 TB = 2^50 B;
1 EByte = 1024 PB = 2^60 B;

2^64 B = 2^60B*2^4 = 16*1EB = 16EB;
